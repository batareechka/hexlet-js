## Установка JavaScript

### Ubuntu или Ubuntu on Windows
```
$ sudo apt-get install curl
$ curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
$ sudo apt install -y nodejs
```

### macOS
```
$ brew install nodejs
```
